# SnovaVerseAnuler Project Overview

## What this project is
SnovaVerseAnuler is an Angular 21 web application for showcasing immersive digital services. The site presents offerings in:

- Virtual Reality (VR)
- AR & AI experiences
- Metaverse solutions
- Interactive tourism/virtual tour experiences

The home page combines marketing content sections (about, services, projects, industries, process, why-us, and call-to-action), with a contact flow and dedicated service pages.

## Core characteristics
- Built with Angular standalone components and Angular Router.
- Supports server-side rendering (SSR) with an Express server entry.
- Includes bilingual behavior (English/Arabic) with direction switching (`ltr` / `rtl`).
- Uses rich media assets (video, images, lottie/3D/web assets) to present services.

## Main user flows
1. User lands on `/home` and sees hero content with project discovery CTA.
2. User navigates to specific service pages (`vr-service`, `AR&AI-service`, `metaverse-service`).
3. User can open the tourism virtual tour page.
4. User reaches `/home/contact-us` to start a business conversation.
